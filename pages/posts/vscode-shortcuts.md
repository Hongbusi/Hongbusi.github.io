## [Keyboard shortcuts for Mac](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf)

| 工作区快捷键            |                          |
| --------------------- | ------------------------ |
| `Cmd` + `Shift` + `P` | 显示命令面板               |
| `Cmd` + `B`           | 显示/隐藏侧边栏            |
| `Cmd` + `\`           | 创建多个编辑器             |
| `Cmd` + `1/2`         | 聚焦到第 1 个、第 2 个编辑器 |
| `Cmd` + `+/-`         | 将工作区放大/缩小           |
| `Cmd` + `J`           | 显示/隐藏控制台            |
| `Cmd` + `Shift` + `N` | 重新开一个软件窗口          |
| `Cmd` + `Shift` + `W` | 关闭软件当前窗口            |
| `Cmd` + `N`           | 新建文件                  |
| `Cmd` + `W`           | 关闭当前文件               |
